# Day 1 - Pre Security Basics

## Topics Covered
- Offensive Security Intro
- Defensive Security Intro
- Careers in Cyber
- Network Fundamentals (Networking, LAN, OSI Model)

## What I Learned
- Cybersecurity has two main sides: offensive (attack) and defensive (protect)
- SOC Analysts monitor systems and respond to threats
- Networks allow devices to communicate with each other
- LAN is a local network (home, office)
- OSI model has 7 layers

## Key Takeaway
Understanding how networks and systems work is essential before learning how to attack or defend them.

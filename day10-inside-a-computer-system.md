# Day 10 - Inside a Computer System

## What I Studied

- Motherboard
- CPU
- RAM
- Storage
- GPU
- PSU
- Boot process

## What I Learned

- The motherboard connects all hardware components.
- The CPU processes instructions and calculations.
- RAM stores temporary working data.
- SSDs are faster than HDDs because they have no moving parts.
- The boot process includes UEFI, POST, boot device selection, and bootloader execution.

## Expanded Understanding

- Firmware is low-level software that initializes hardware before the operating system loads.
- UEFI replaced the older BIOS system in most modern computers.
- RAM is volatile memory, meaning data is lost when power is removed.
- Cybersecurity attacks can target firmware, bootloaders, memory, and storage devices.

## Key Takeaway

Understanding how computer components interact is important because cybersecurity involves protecting systems at multiple levels, including hardware, firmware, operating systems, memory, and networks.
